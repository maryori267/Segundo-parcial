¿Qué son los contenedores?

se trata de elementos que agrupan o encierran contenido para organizarlo, estructurarlo o darle estilo.

Son bloques que se usan para:

Agrupar texto, imágenes, formularios, etc. Aplicar estilos como márgenes, fondo, bordes, etc. Controlar el diseño (por ejemplo: en columnas o filas).

¿Cuándo se debe usar un contenedor (container)?

Debes usar un contenedor cuando quieres:
1. Centrar el contenido en la página Los contenedores en Bootstrap colocan el contenido con márgenes automáticos a los lados para que se vea bien en diferentes tamaños de pantalla.
2. Limitar el ancho del contenido Un container tiene un ancho máximo definido para que el contenido no se vea muy estirado en pantallas grandes. Un container-fluid ocupa todo el ancho de la pantalla.
3. Organizar filas y columnas Para que las clases row y col funcionen correctamente, deben estar dentro de un container.
4. Aplicar diseño responsivo fácilmente Usar container ayuda a que el sitio se adapte mejor en celular, tablet y computadora.
